# ALICE-Proxy

ALICEエコシステム向けのL7リバースプロキシエンジン。ルーティング、負荷分散、ヘッダー/パス書き換え、サーキットブレーカー、レート制限、ヘルスチェックを純Rustで提供。

## 機能

- **ルーティング** -- パスマッチング(完全一致、プレフィックス、ワイルドカード、正規表現)、ホストマッチング、メソッドマッチング
- **負荷分散** -- ラウンドロビン、ランダム、最少接続、IPハッシュ戦略(重み付きアップストリーム対応)
- **ヘッダー書き換え** -- ヘッダーの設定・追加・削除、リクエスト/レスポンス用チェーンルール
- **パス書き換え** -- プレフィックス除去・追加、正規表現置換
- **サーキットブレーカー** -- 障害閾値、タイムアウト、ハーフオープンプローブ(アップストリーム別レジストリ)
- **レート制限** -- トークンバケットアルゴリズム(容量・補充レート設定可能)
- **ヘルス追跡** -- アップストリーム別ヘルスステータス(Healthy / Degraded / Unhealthy)
- **リクエスト/レスポンス変換** -- ボディサイズ制限、メソッドオーバーライド、ステータス書き換え、CORS注入
- **リトライポリシー** -- リトライ可能ステータスコード付きの最大リトライ回数設定

## アーキテクチャ

```
Request --> Router (パス + ホスト + メソッドマッチング)
              |
              v
         Route --> LoadBalancer (ラウンドロビン / ハッシュ / 最少接続)
              |
              v
         RequestTransform --> HeaderRewrite --> PathRewrite
              |
              v
         CircuitBreaker --> アップストリーム選択
              |
              v
         ResponseTransform --> Response
```
